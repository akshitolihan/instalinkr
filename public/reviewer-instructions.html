Reviewer Instructions

This page explains the InstaLinkr App Review flow for Meta reviewers.

App URL

https://instalinkr.com/login

Test Flow

  1. Open the app URL and create an account with email/password, or continue with Facebook.
  2. After sign-in, open Settings and choose Connect Instagram.
  3. Authorize the Facebook account that manages an Instagram professional account linked to a Facebook Page.
  4. Show the connected Instagram account and Facebook Page in Settings so the selected asset is visible.
  5. Return to the dashboard and create a campaign with a keyword, optional Instagram post URL, delivery link, and message template.
  6. For the App Review screencast, comment on the connected Instagram account's own media, then use Settings > Live review test to send a private reply to the real Instagram comment ID.
  7. Open the native Instagram app or website and show the same private reply delivered to the commenter.
  8. In normal production use, when someone comments the configured keyword on the connected Instagram account's own media, the webhook routes the comment to that account and sends a private reply containing the configured link.
  9. The lead appears in the Leads table, where the account owner can edit email/follower status or export CSV.

Screencast Checklist

  • Show the complete Meta login and permission grant flow.
  • Show the selected Instagram account and Facebook Page asset in the app.
  • Show the live private reply send action in Settings > Live review test.
  • Show the delivered message in the native Instagram client after sending.

Permission Usage

  • instagram_basic: displays connected Instagram account ID and username so the account owner knows which account is connected.
  • instagram_manage_comments: receives comment webhook events and reads comment IDs so campaign keywords can match comments.
  • instagram_manage_messages: sends the configured private reply to the commenter using the comment ID.

Notes

The app does not ask for or expose commenter follower status from Meta because the official API does not provide a direct follower check for arbitrary commenters. The optional follower field is self-reported or manually verified by the account owner.
